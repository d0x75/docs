Operadores Aritiméticos
-------------------------

**Atribuição**: =

	a = 2
	print(a)


**Adição**: +
		
	a = 2
	print(a + 10)


**Subtração**: -
		
	a = 2
	print(a - 10)


**Multiplicação**: *
	
	a = 2
	print(a * 10)


**Divisão**: / 

	a = 20
	print(a / 10)


**Módulo**: %

	a = 30
	print(a % 10)


**Potenciação** : **

	a = 5
	print(a ** 2)
